  • It helps to convert any image to pixel art.
Pixatool Workflow 01: Picked Palette

Follow this sequence:

  • 0: Prepare the image: resize, optimize contrast and saturation, evaluate posterization/color reduction.
  • 1: Import the image
  • 2: Left Sidebar Palette -> Clear
  • 3: Palette -> New
  • 4: click color position and use Pick on all the main colors
  • 5: Palette -> Set to convert the colors
  • 6: Iterate eventually back to step 4 to add more hues
  • 7: Proceed with General for pixelate H/W, Contrast, Basic Blur etc.

Follow this sequence:

  • 0: move chosen image from 'originals' folder to 'published'
  • 1: load image in photomania editor or animeGANv2
  • 2: choose filter: Painting -> Cartoonizer
  • 3: save to 'published' with prefix -> DnDe-00n-orig-
  • 4: pixatool settings: 6 px horizontal, 6 px vertical, blur: 3, contrast: 120, exposure: 32, dither: 6
  • 5: duplicate layer (CTRL+j) and set blending mode to color; posterize: 5/6
  • 6: scale 600%
  • 7: save to 'published' folder
