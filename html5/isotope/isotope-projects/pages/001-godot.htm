Pixatool Workflow 02: Iconic

Image sources:

  • @pixelart src-movie GIFs
  • @pixelart STYLE

Follow this sequence:

  • 0: Prepare the image: resize, optimize contrast and saturation, evaluate posterization/color reduction.
  • 1: Import the image
  • 2: Left Sidebar Palette -> Clear
  • 3: Palette -> New
  • 4: click color position and use Pick on all the main colors
  • 5: Palette -> Set to convert the colors
  • 6: Iterate eventually back to step 4 to add more hues
  • 7: Proceed with General for pixelate H/W, Contrast, Basic Blur etc.
Pixatool Workflow 03: Pixel Lands

Move new image chosen from Image Source:

    /home/masayume/Downloads/masayumeP/DONE/LANDS-READY/NEW/

To Image Destination:

    /home/masayume/Downloads/masayumeP/DONE/LANDS-READY/PUB/published.2/

Follow this sequence:

  • 0: edit new image in Photopea
  • 1: CTRL+U +20 saturation
  • 2: CTRL+J duplicate layer
  • 3: set new "layer style" to: Color
  • 4: choose "image adjustment": Posterize
  • 5: add new image to another layer in /home/masayume/Downloads/masayumeP/DONE/LANDS-READY/PUB/_02-serie-template.psd
  • 6: edit new layer image number
  • 7: save new _02-serie-template.psd and move to /home/masayume/Downloads/masayumeP/DONE/LANDS-READY/PUB/
