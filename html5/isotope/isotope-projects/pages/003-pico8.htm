  • 0: Prepare the image: resize, optimize contrast and saturation, evaluate posterization/color reduction.
  • 1: Import the image

Follow this sequence:

  • 0: edit new image in Photopea
  • 1: CTRL+U +20 saturation
  • 2: CTRL+J duplicate layer
  • 3: set new "layer style" to: Color
  • 4: choose "image adjustment": Posterize
  • 5: add new image to another layer in /home/masayume/Downloads/masayumeP/DONE/LANDS-READY/PUB/_02-serie-template.psd
  • 6: edit new layer image number
  • 7: save new _02-serie-template.psd and move to /home/masayume/Downloads/masayumeP/DONE/LANDS-READY/PUB/
